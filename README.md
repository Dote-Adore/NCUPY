# NCUPY
## 简介
这是个人第一次制作一个完整的项目，也是第一次制作web前端，里面的内容也许不是特别好，命名或者是函数也不是特别规范，还请见谅
本项目是校内二手交易平台微信小程序，鉴于校内很多人需要卖二手而没有一个特别好的平台而制作，能不能真的在校内推广我也不能把握。
由于前端与后端同时进行开发，接口也是需要使用的时候再写，之前也没有有一个特别好的项目规划，可能整个项目看起来比较乱，之后我也会
将后端代码一并上传，
如果有什么问题，欢迎与我交流~<br>
